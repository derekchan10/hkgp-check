# 中签结果比对工具

一个纯前端 Web 工具，用于比对账号信息与中签结果。文件在本地浏览器内处理，不上传任何数据。

**在线使用：http://dcdddd.com/hkgp-check/**

## 功能特点

- 支持 Excel 格式（.xlsx, .xls）的文件
- 自动提取账号后三位
- 自动提取姓名的前 N 个英文字母（自动检测长度）
- 中文姓名自动转换拼音，支持多音字特殊处理
- 显示所有账号匹配状态及数量统计
- 支持结果导出为 CSV 格式
- 纯静态页面，无需服务器

## 文件格式

**账号文件**（两列，第一行为表头）

| 姓名 | 账号 |
|---|---|
| 张三 | 123456789 |

**中签结果文件**（四列，第一行为表头）

| 账号后三位 | 名字头5个字母 | 投注数 | 中签数 |
|---|---|---|---|
| 789 | ZHANG | 100 | 5 |

## 匹配逻辑

- **账号**：提取后三位进行精确匹配
- **姓名**：
  - 英文姓名 → 提取字母，大写，截取前 N 位
  - 中文姓名 → 转拼音，大写，截取前 N 位
  - N 由结果文件自动检测（取最常见长度）

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何安装。
